# 麦氏轮PID控制学习指南

> 从零开始，循序渐进掌握麦氏轮机器人的精确控制

---

## 为什么学习这个？

四轮麦氏轮机器人是嵌入式控制领域的经典项目。通过学习它的控制方法，你将掌握：

- **编码器测速原理** - 如何知道轮子转多快
- **PID控制算法** - 如何让轮子达到目标速度
- **运动学解算** - 四个轮子如何协同工作
- **双层闭环控制** - 内环外环的分工与配合

这些知识是机器人控制的基础，掌握后可迁移到其他项目。

## 核心概念预览

在开始学习前，先建立整体认知：

### 我们要控制什么？

四轮麦氏轮机器人有三个独立的运动自由度：

| 运动类型 | 符号 | 说明 |
|---------|------|------|
| 前进/后退 | Vx | 沿机器人前后方向移动 |
| 左右平移 | Vy | 沿机器人左右方向移动 |
| 原地旋转 | ω | 绕机器人中心旋转 |

**控制目标**：让机器人按照期望的 Vx、Vy、ω 运动。

### 控制架构

```
用户输入：target_vx, target_vy, target_ω
              ↓
         外环PID（决策层）
              ↓
    运动学解算 → target_rpm[4]
              ↓
         内环PID（执行层）
              ↓
         PWM输出 → 电机转动
              ↓
         编码器反馈 → actual_rpm[4]
              ↓
    逆运动学解算 → current_vx, current_vy, current_ω
              ↓
         反馈到外环
```

**核心思想**：
- 内环让轮子达到某个转速
- 外环决定这个转速应该是多少
