# 三、麦氏轮运动学

> 理解四个轮子的协同关系

---

## 模块概述

本模块讲解麦氏轮的运动学原理，包括：

- **麦氏轮结构**：为什么能全向移动
- **正向运动学**：机器人速度 → 轮子速度
- **逆向运动学**：轮子速度 → 机器人速度

运动学是四轮协同控制的数学基础。

---

## 学习目标

完成本模块学习后，你将能够：

1. 理解麦氏轮的全向移动原理
2. 掌握正向运动学公式
3. 掌握逆向运动学公式
4. 理解典型运动模式的轮速配合

## 核心问题

在开始学习前，先思考一个问题：

**四个轮子如何配合才能让机器人按期望运动？**

麦氏轮机器人的三个运动自由度：
- 前进/后退（Vx）
- 左右平移（Vy）
- 原地旋转（ω）

每个运动都需要四个轮子以特定方式配合。
