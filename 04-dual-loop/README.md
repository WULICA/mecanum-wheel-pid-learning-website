# 四、双层闭环控制

> 理解内环外环的架构设计

---

## 模块概述

本模块讲解双层闭环控制的架构设计，包括：

- **内环控制**：单轮速度PID
- **外环控制**：机器人整体速度PID
- **数据流**：内外环如何配合

双层闭环是麦氏轮精确控制的核心架构。

---

## 学习目标

完成本模块学习后，你将能够：

1. 理解为什么需要双层控制
2. 掌握内环外环的分工
3. 理解内外环的数据传递
4. 掌握内外环的参数整定方法

## 核心问题

在开始学习前，先思考一个问题：

**为什么不能只用内环控制？**

如果只控制每个轮子的转速，机器人整体速度可能不准确：
- 轮子打滑
- 负载不均
- 地面摩擦差异

**双层控制**：内环管轮子，外环管机器人，协同工作。
