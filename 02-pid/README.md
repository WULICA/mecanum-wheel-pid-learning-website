## 模块概述

本模块讲解PID控制的核心原理，包括：

- **PID本质**：我们要解决什么问题

PID是最经典、最实用的控制算法，理解它是学习控制的基础。

## 核心问题

在开始学习前，先思考一个问题：

**如何让电机达到并保持目标转速？**

假设你希望电机以100 RPM运行，但实际只有85 RPM。原因可能是：
- 负载变化
- 摩擦力变化
- 电压波动

**PID的任务**：不断调整PWM占空比，让实际转速接近目标转速。
