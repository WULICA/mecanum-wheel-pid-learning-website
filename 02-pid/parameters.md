# 参数整定方法

> 如何调整PID参数

---

## 整定原则

### 基本思路

```
1. 先调Kp：让系统有响应
2. 再调Ki：消除静差
3. 最后Kd：抑制超调（如果需要）
```

### 口诀

```
先比例，后积分，再微分
幅值大，比例小
有静差，积分加
有超调，微分减
```

---

## 整定步骤

### 步骤1：调Kp

**目标**：让系统有响应，但不振荡

```
方法：
1. Ki = 0, Kd = 0
2. 从小到大增加Kp
3. 观察响应曲线

现象：
  Kp太小：响应慢，达不到目标
  Kp适中：响应快，无振荡
  Kp太大：开始振荡
  Kp过大：持续振荡或发散

建议：找到开始振荡的Kp，取其一半
```

### 步骤2：调Ki

**目标**：消除静差，但不引起振荡

```
方法：
1. 保持Kp不变
2. 从小到大增加Ki
3. 观察稳态误差

现象：
  Ki太小：有静差
  Ki适中：无静差，稳定
  Ki太大：开始振荡

建议：缓慢增加，直到静差消除
```

### 步骤3：调Kd

**目标**：抑制超调（如果需要）

```
方法：
1. 保持Kp、Ki不变
2. 从小到大增加Kd
3. 观察超调量

现象：
  Kd太小：有超调
  Kd适中：超调减小
  Kd太大：响应变慢，可能抖动

建议：速度环通常不用Kd
```

## 典型参数范围

### 速度环PID

| 参数 | 典型范围 | 说明 |
|------|---------|------|
| Kp | 0.5 ~ 3.0 | 根据电机特性调整 |
| Ki | 0.01 ~ 0.2 | 不宜过大 |
| Kd | 0 ~ 0.1 | 通常不用 |

### 位置环PID

| 参数 | 典型范围 | 说明 |
|------|---------|------|
| Kp | 1.0 ~ 10.0 | 比速度环大 |
| Ki | 0.01 ~ 0.5 | 消除位置静差 |
| Kd | 0.1 ~ 1.0 | 抑制位置超调 |

## 调试方法

### 方法1：串口观察

```c
// 在PID计算后打印
printf("target: %.2f, actual: %.2f, error: %.2f, output: %.2f\n",
       target_rpm, actual_rpm, error, pwm_output);
```

### 方法2：波形观察

使用串口绘图工具（如VOFA+、SerialPlot）实时绘制曲线：

```
目标曲线：一条水平线
实际曲线：应该快速接近目标线
误差曲线：应该快速收敛到0
```

### 方法3：阶跃响应测试

```
1. 给定一个阶跃目标（如从0跳到100 RPM）
2. 观察响应曲线
3. 分析：上升时间、超调量、调节时间
```

## 常见问题与解决

### 问题1：响应太慢

**现象**：实际值缓慢接近目标值

**原因**：Kp太小

**解决**：增大Kp

### 问题2：有静差

**现象**：实际值稳定在目标值附近，但不等于目标值

**原因**：Ki太小或为0

**解决**：增大Ki

### 问题3：振荡

**现象**：实际值在目标值上下波动

**原因**：Kp或Ki太大

**解决**：减小Kp和Ki

### 问题4：超调大

**现象**：实际值先超过目标值，再回落

**原因**：Kp太大，或没有D项

**解决**：减小Kp，或增加Kd

### 问题5：抖动

**现象**：实际值快速小幅波动

**原因**：Kd太大，或测量噪声大

**解决**：减小Kd，或对测量值滤波

## 参数整定示例

### 示例：速度环PID整定

**初始状态**：
```
Kp = 0, Ki = 0, Kd = 0
目标转速：100 RPM
```

**步骤1：调Kp**
```
Kp = 0.5：响应慢，有静差（实际95 RPM）
Kp = 1.0：响应较快，有静差（实际98 RPM）
Kp = 2.0：响应快，开始振荡
Kp = 1.0：选择这个值
```

**步骤2：调Ki**
```
Ki = 0.01：静差减小（实际99 RPM）
Ki = 0.05：无静差，稳定
Ki = 0.1：开始振荡
Ki = 0.05：选择这个值
```

**最终参数**：
```
Kp = 1.0
Ki = 0.05
Kd = 0（速度环不用）
```

## 不同电机的参数差异

### 影响因素

| 因素 | 影响 |
|------|------|
| 电机功率 | 功率大，Kp可以小 |
| 负载惯量 | 惯量大，响应慢，Kp要大 |
| 减速比 | 减速比大，Kp要小 |
| 编码器分辨率 | 分辨率高，Ki可以小 |

### 建议

```
1. 每个电机单独调参
2. 同型号电机参数相近
3. 记录调好的参数，方便复用
```

## 自动整定方法

### Ziegler-Nichols方法

**步骤**：
1. Ki = 0, Kd = 0
2. 增大Kp直到系统开始持续振荡
3. 记录临界Kp（Ku）和振荡周期（Tu）
4. 查表计算参数

**PI控制器参数**：
```
Kp = 0.45 × Ku
Ki = 0.54 × Ku / Tu
```

**注意**：此方法可能导致系统不稳定，谨慎使用。

---

## 本节要点

1. **整定顺序**：先Kp，再Ki，最后Kd
2. **速度环**：通常用PI，不用D
3. **观察方法**：串口打印、波形绘制
4. **常见问题**：响应慢、静差、振荡、超调
5. **参数记录**：调好后要记录，方便复用
