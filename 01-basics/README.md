# 一、基础概念

> 建立对编码器和速度测量的基本认识

---

## 模块概述

本模块讲解麦氏轮PID控制的基础知识，包括：

- **编码器原理**：增量式编码器如何输出信号
- **速度计算**：从编码器脉冲数到电机转速的换算
- **STM32配置**：如何用定时器读取编码器

这些是后续PID控制的基础，务必理解透彻。

---

## 学习目标

完成本模块学习后，你将能够：

1. 理解增量式编码器的工作原理
2. 掌握速度计算公式的推导过程
3. 在STM32上配置定时器编码器模式
4. 读取并计算电机的实时转速

## 核心问题

在开始学习前，先思考一个问题：

**我们怎么知道轮子转多快？**

这是所有速度控制的基础。如果不能准确测量当前速度，就无法进行闭环控制。

编码器就是解决这个问题的传感器。
